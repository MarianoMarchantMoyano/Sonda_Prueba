AUTOMATIZACION SELEIUM IDE - SONDA

Link Repositorio:

https://github.com/MarianoMarchantMoyano/Sonda_Prueba.git

Descripcion:

Se ocupo la herramienta Selenium IDE para poder arrancar la prueba para sonda.



Instalacion Repositorio:

1.- Abrimos CMD en la carpeta donde queremos nuestro proyecto
2.- En consola colocaremos el comando : git clone https://github.com/MarianoMarchantMoyano/Sonda_Prueba.git




Instalacion Selenium IDE:


1.- Abrimos chrome y en la URL colocaremos https://www.selenium.dev/selenium-ide/
2.- hacemos clic en CHORME DOWNLOAD 
3.- Luego nos redirigira a https://chrome.google.com/webstore/detail/selenium-ide/mooikfkahbdckldjjndioackbalphokd
    donde instalremos la extencion de SELENIUM IDE.
4.- Luego al lado del recuadro donde colocamos las URL nos saldra la extencion de SELENIUM IDE le haremos clic. 
5.- Se nos abrira la ventan nueva con SELENUIM corriendo de forma correcta.


Uso de Selenium IDE recien instalado:

1.- Listo con la ventana de SELENIUM haremos clic en CREATE A NEW PROJECT 
2.- Luego colocaremos en PROJECT NAME el nombre de nuestro proyecto le daremos OK.
3.- Luego en nuestra izquiera de nuestra pantalla en SELENUIM nos saldra un TEST llamado UNTITLED. 
4.- Para cambiar el name , haremos clic en 3 puntos donde dice UNTITLED y colocaremos RENAME colocaremos sonda_1 como name.
5.- Luego donde dice PLAYBACK BASE URL colocaremos nuestro URL en donde haremos pruebas atomatizadas como por EJ : https://demoqa.com/
6.- Haremos clic en REC que esta al lado derecho de neustro programa.
7.- ya dentro de nuestra paguina haremos las pruebas necesarias para nuestro nuevo poryecto.
8.- Listo con las pruebas vamos a SELENUIM IDE colocaremos STOP al REC para poder visualizar lo que queremos probar en nuestra WEB
9.- para esto nos iremos hacia la barra que esta arriba del URL, colocaremos test execution speed para ver a que velocidad queremos
    que corra nuestra prueba de automatizacion.
10.- Luego haremos clic en RUN CURRENT TEST en donde correra nuestro proyecto para hacer un testeo completo de la web y revisar 
     errores que puedan ocurrir.


Uso de Selenium IDE ya instalado:

1.- Listo con nuestro SELENIUM IDE instalado , le haremos clic en nuestra extencion 
2.- Luego haremos clic en OPEN AN EXISTING PROJECT 
3.- Esto no abrira una ruta en donde colocaremos nuestro proyecto ya existente llamado Sonda_prueba
4.- Para correr nuestro proyecto fijarnos que en test execution speed dejarlo en fast asi poder correr el proyecto sin pausas de 
    tiempo , si colocamos mas lento nuestro TEST puede tardar mas de lo devido.
